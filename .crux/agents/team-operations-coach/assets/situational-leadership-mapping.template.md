# Situational Leadership Mapping

This reference helps `team-operations-coach` convert natural management answers
into internal `G`, `S`, and `T` signals.

## Purpose

Users should answer in plain language.
The agent can map those answers to internal shorthand for consistency.

## Growth Levels

- `G1` — new or inexperienced in this scope; needs clear direction
- `G2` — developing but inconsistent; needs both direction and encouragement
- `G3` — capable but unevenly confident or unevenly aligned; benefits from support
- `G4` — highly capable and reliable in this scope; can work with broad delegation

## Support Styles

- `S1` — directing
- `S2` — coaching
- `S3` — supporting
- `S4` — delegating

## Team Levels

- `T1` — forming or unstable team rhythm
- `T2` — basic rhythm exists, but ownership or visibility is uneven
- `T3` — stable rhythm, clear ownership, dependable weekly updates
- `T4` — mature, autonomous, and improving team rhythm

## Natural Language Clues

### Autonomy

- "needs close direction" → lower autonomy
- "mixed" → middle range
- "works independently" → higher autonomy

### Leadership Style

- "mostly directs" → `S1`
- "directs and explains" or "coaches through it" → `S2`
- "supports and unblocks" → `S3`
- "delegates heavily" → `S4`

### Team Maturity

- "still forming" → `T1`
- "growing but not steady yet" → `T2`
- "steady rhythm" → `T3`
- "self-managing and improving" → `T4`

## Usage Notes

- Do not present the code as the primary answer
- Prefer plain language in weekly and leadership reports
- Use the code internally for consistent interpretation and lightweight metadata
- Re-map if the organisation's cadence or structure changes materially
