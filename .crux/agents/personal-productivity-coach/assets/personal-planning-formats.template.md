# Personal Planning Formats

Crux should support simple markdown planning formats that are easy to reuse.

---

## Daily Plan

Recommended structure:
- Top Priorities
- Quick Wins
- Waiting / Follow-Up
- If Time Allows

---

## Weekly Review

Recommended structure:
- Wins
- Open Loops
- Waiting Items
- Next Week Priorities
- Notes to Archive or Move

---

## Output Style Guidance

- `checklist` → short action-heavy format
- `short-note` → brief narrative plus bullets
- `daily-plan` → default practical planning format

Use the shortest format that still helps the user act.
